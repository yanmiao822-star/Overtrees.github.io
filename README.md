# 此刻 — 一个轻量的 iOS 本地笔记

**记录此刻，无需等待。**

「此刻」是一个为 iOS 打造的本地优先笔记应用。没有账号、没有同步焦虑、不需要网络——打开即写，数据全在本地。

---

## UI / 界面

**iOS 原生感，不是 Web App 感。**

- **毛玻璃菜单** — 所有弹层和菜单使用 `backdrop-filter: blur(32px)`，与 iOS 控制中心一致的质感
- **Squircle 圆角** — 卡片、按钮、图片使用连续曲率圆角，比普通 `border-radius` 更柔和（iOS 17.4+）
- **动态标题栏** — 列表页大标题随滚动平滑收缩为居中 pill 样式，类似 Apple Music
- **暗色模式** — 完整支持 `prefers-color-scheme: dark`，黑白两套变量体系
- **安全区适配** — 全面使用 `env(safe-area-inset-*)`，刘海/岛/底部横条完整适配
- **状态栏毛玻璃** — 顶部渐变遮罩，内容滚动时状态栏不突兀

**视觉风格：**
- 纯白/纯黑底色，无分割线阴影，用 0.5px 边框替代
- 卡片圆角 32px / 22px / 999px 三级体系，视觉层级清晰
- 图标全部使用 SF Symbols 风格的线性 SVG

## 交互

**触感优先，键盘其次。**

- **三指撤销/重做** — iOS 原生三指手势，左右滑动即可撤销/重做，与外接键盘 Cmd+Z 并存
- **左滑操作** — 笔记列表左滑露出「置顶」和「删除」，跟随手指位移的弹性动画
- **斜杠命令** — 在编辑器中输入 `/` 触发命令面板，支持拼音首字母搜索（如 `bt`=标题、`tp`=图片）
- **智能前缀** — 输入 `# ` → 标题、`> ` → 引用、`[ ] ` → 待办、`---` → 分割线，即时转换
- **卡片点击防误触** — 链接卡片左右边缘 24px 内不响应跳转，避免编辑时误触
- **光标跟随滚动** — 编辑长标题时自动滚动保持光标可见
- **图片取色** — 列表卡片自动从首张图片中提取主色调作为背景底色

## 设计理念

- **内容即界面** — 编辑器没有工具栏，通过「打字 + 前缀/斜杠」驱动所有操作
- **渐进呈现** — 不一次性展示所有功能，用户打字过程中自然发现（输入 `#` 出现标题，输入 `/` 出现面板）
- **可预览的 Markdown** — 不是 split-pane 编辑/预览分离，而是编辑器中直接渲染，选中段落可切回编辑态
- **离线的第一公民** — 所有核心功能（创建/编辑/搜索/删除）不依赖网络，网络只是增强（链接卡片抓取）
- **数据透明** — 没有隐藏的数据库格式，数据存 IndexedDB，可导出为 ZIP，用户始终拥有完整控制权

## 可玩性

**它不是一个"写完了就放着"的笔记工具，而是一个可以玩的东西。**

- **块类型组合** — 段落 + 图片画廊 + 音乐卡片 + 待办列表，可以拼出丰富的笔记页面
- **内联装饰语法** — `==高亮==`、`~~删除线~~`、`!!注意`、`??疑问`、`@@提及`、`$标签$`、`#话题#`，输入即渲染
- **Callout 提示块** — `> [!NOTE]` / `> [!WARNING]` / `> [!TIP]` / `> [!IMPORTANT]` / `> [!CAUTION]`，五种风格
- **装饰分割线** — `---emoji---` 产生带 emoji 的装饰分割线
- **居中语法** — `->文字<-` 生成居中文本
- **彩色标签** — `$red:重要$` 或 `$蓝色标签$` 生成彩色药丸标签
- **Emoji 短代码** — `:smile:` → 😊、`:fire:` → 🔥、`:rocket:` → 🚀，30+ 常用 emoji
- **卡片系统的发现乐趣** — 粘贴音乐链接 → 自动变成带封面的音乐卡片；粘贴普通 URL → 变成链接卡片；粘贴纯图片 URL → 变成图片块

**可玩性的核心：输入内容 → 系统自动美化。** 你只管写，剩下的交给它。

## 它能解决什么

- **想记点东西但 app 启动太慢** → 开屏即写，0 等待
- **笔记散落在各个平台** → 一个纯本地文件，数据可导出、可备份、永远属于你
- **依赖网络才能用的笔记工具** → 断网也能编辑、保存、查看全部笔记
- **数据被厂商锁定** → 纯前端 + IndexedDB + ZIP 导出，随时可以拿走
- **编辑器太重** → 没有复杂的工具栏/菜单/排版引擎，一个 contentEditable 搞定

## 功能全景

### 编辑器
- 块类型：段落、标题 H1-H3、待办、引用、分割线、装饰分割线
- 卡片类型：音乐卡片、链接卡片、图片卡片、图文卡片、图文行、图片画廊
- 斜杠命令面板：20+ 命令、拼音首字母搜索、最近使用
- 智能前缀：`#` → H1、`##` → H2、`###` → H3、`>` → 引用、`[ ]` → 待办、`---` → 分割线
- Markdown 自动渲染：表格、Callout、代码块、列表、引用、删除线、高亮、标签
- 撤销/重做：三指手势、Cmd+Z/Y、菜单按钮

### 存储
- IndexedDB 主存储（笔记 + 附件 + 渲染缓存）
- localStorage 双重写入备份
- Google Drive 云端备份/恢复（覆盖 + 合并）
- 本地 ZIP 导出/导入（含附件）

### 列表
- 时间分组：今天 / 昨天 / 过去 7 天 / 过去 30 天 / 月份
- 置顶笔记
- 图片缩略图自动提取
- 标签自动提取和显示
- 展开/折叠长笔记

### 个性化
- 字体大小：小/中/大三档
- 笔记专属背景色：8 种预设色
- 笔记头像：自定义图片

## 技术栈

- **React 18**（UMD，无构建工具）
- **contentEditable** 块编辑器
- **IndexedDB** + **localStorage** 分层存储
- **Cloudflare Worker**（链接元数据抓取 + 图片代理）
- **Google Drive API**（可选云备份）
- **fflate**（ZIP 压缩）
- **marked** + **DOMPurify**（MD 渲染安全过滤）
- 全部依赖本地化，无运行时 CDN 外链

## 架构

```
Moments.html        ← 主应用（6,500 行，壳层）
  src/
    constants.js          ← API 端点、正则、配置
    utils/                ← ID/时间/DOM/SVG 工具
    services/             ← IndexedDB + localStorage + 备份
    blocks/               ← 13 种块类型注册
    editor/               ← MD 渲染 + 编辑器核心
    components/           ← UI 组件
    hooks/                ← 手势 Hook
  vendor/               ← React/marked/DOMPurify 等本地依赖
```
