# 此刻 — 一个轻量的 iOS 本地笔记

**记录此刻，无需等待。**

「此刻」是一个为 iOS 打造的本地优先笔记应用。没有账号、没有同步焦虑、不需要网络——打开即写，数据全在本地。

## 它能解决什么

- **想记点东西但 app 启动太慢** → 开屏即写，0 等待
- **笔记散落在各个平台** → 一个纯本地文件，数据可导出、可备份、永远属于你
- **依赖网络才能用的笔记工具** → 断网也能编辑、保存、查看全部笔记
- **数据被厂商锁定** → 纯前端 + IndexedDB + ZIP 导出，随时可以拿走

## 功能

- 📝 **块编辑器** — 段落、标题、待办、引用、分割线、图片画廊
- 🎵 **智能卡片** — 粘贴音乐链接（网易云/QQ音乐/Spotify）自动识别为音乐卡片
- 🔗 **链接卡片** — 粘贴 URL 自动抓取标题、封面、来源
- ✍️ **Markdown 自动渲染** — 表格、Callout 提示块、高亮、标签、代码块都在输入后自动呈现
- /️⃣ **斜杠命令面板** — 输入 `/` 快速插入任意块类型，支持拼音搜索
- ↩️ **撤销/重做** — iOS 三指手势 + 键盘快捷键 + 菜单按钮
- 🖼 **图片附件** — 插入图片自动生成缩略图，原图存 IndexedDB
- 🏷 **标签系统** — 内联 `#标签` 语法，列表自动提取
- 📁 **分组浏览** — 按时间自动分组（今天/昨天/7天/30天/月份）
- 🎨 **纯色背景** — 为每条笔记设置专属底色
- 🔄 **Google Drive 备份** — 云端备份/恢复，支持覆盖和合并
- 📥 **本地 ZIP 导出/导入** — 数据随时可迁移
- 🌙 **深色模式** — 跟随系统

## 特色

- **iOS 原生体验** — 全面适配 `safe-area-inset`、毛玻璃效果、squircle 圆角、触感反馈
- **离线优先** — 所有核心依赖本地化（React/marked/DOMPurify 等），断网完整可用
- **单页应用** — 无框架构建，纯原生 JS + React UMD
- **数据安全** — IndexedDB 主存储 + localStorage 双重写入，不怕丢失
- **隐私优先** — 笔记不经过任何服务器，除非你主动使用云备份
- **轻量部署** — 纯静态文件托管（GitHub Pages），一个仓库即可

## 技术栈

- React 18（UMD，无构建工具）
- contentEditable 块编辑器
- IndexedDB + localStorage 分层存储
- Cloudflare Worker（链接元数据抓取）
- 纯 CSS 变量主题系统

## 快速开始

直接打开：https://overtrees.github.io/Mynote/

部署到自己的 GitHub Pages：

```bash
git clone https://github.com/你的用户名/你的仓库.git
# 将 Moments.html + src/ + vendor/ + index.html 放入根目录
git push
```

数据完全在浏览器本地，部署只是托管静态文件。
