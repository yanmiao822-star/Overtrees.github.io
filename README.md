# 此刻

一个在 iOS 上很好用的本地笔记。打开就能写，不用联网，不想要账号。

---

## 长什么样

**看着像 iOS 原生应用，不是那种"一眼网页"的感觉。**

- 菜单和弹层都是毛玻璃效果（`backdrop-filter: blur(32px)`），跟 iOS 控制中心一个质感
- 卡片圆角用了 squircle 连续曲率，比普通圆角更柔和（iOS 17.4+ 上有效果）
- 列表页的标题会随着滚动从大标题平滑缩成一个居中药丸，类似 Apple Music 那样的效果
- 深色模式完整支持，跟系统设置走
- 刘海、灵动岛、底部横条都适配好了，不会挡内容
- 状态栏位置有一层渐变遮罩，内容滚动时不会觉得突兀
- 整体配色走极简路线：纯白/纯黑底，用 0.5px 细线代替分割线，没有多余阴影

---

## 怎么用

**主要是触屏操作，键盘是附加的。**

- **三指撤销/重做** — iOS 原生手势，左滑撤销、右滑重做。外接键盘也能用 Cmd+Z / Shift+Cmd+Z
- **列表左滑** — 划出「置顶」和「删除」，动画跟手
- **斜杠命令** — 编辑器里打 `/` 弹出命令面板，支持拼音首字母搜索（`bt` → 标题、`tp` → 图片）
- **智能前缀** — 输入 `# ` 变标题、`> ` 变引用、`[ ] ` 变待办、`---` 变分割线，回车即生效
- **卡片防误触** — 链接卡片左右边缘 24px 内不响应跳转，编辑时不会误点
- **图片取色** — 列表卡片会从第一张图片里取主色调作为背景底色，每张卡片颜色不一样

---

## 核心理念

- **打字驱动一切** — 没有工具栏，没有菜单栏。输入 `#` 出标题，输入 `/` 出命令面板，输入 `[ ]` 出待办。一切都是打字过程中的自然发现
- **Markdown 所见即所得** — 不是 split-pane 那种左边写右边预览，而是编辑器里直接渲染渲染好，点一下就能切回去编辑
- **离线是默认状态** — 创建、编辑、删除、搜索全部不需要网络。网络只用来抓链接卡片的信息
- **数据你的** — 存 IndexedDB，可以导出 ZIP，随时能拿走。没有隐藏格式、没有厂商锁定

## 可以玩什么

**它不是那种"写完了就放着"的工具，有些东西还挺好玩的。**

- **拼积木一样的块** — 段落 + 图片 + 音乐卡片 + 待办列表 + 画廊，可以拼出很丰富的页面
- **内联装饰** — `==高亮==`、`~~删除线~~`、`!!注意`、`??疑问`、`@@提及`、`$标签$`、`#话题#`，输入完自动渲染
- **Callout 提示块** — `> [!NOTE]` / `> [!WARNING]` / `> [!TIP]` 五种风格，写注意事项很好看
- **装饰分割线** — `---🌟---` 产生带 emoji 的分割线
- **居中文字** — `->文字<-` 居中对齐
- **彩色标签** — `$red:重要$` 或 `$蓝色标签$` 生成彩色药丸标签
- **Emoji 快捷输入** — `:smile:` → 😊、`:fire:` → 🔥、`:rocket:` → 🚀，30 多个常用 emoji
- **卡片彩蛋** — 贴个网易云/QQ音乐/Spotify 链接 → 自动变音乐卡片带封面。贴个普通网址 → 变链接卡片带标题和摘要。贴个纯图片链接 → 变成图片块。这些东西都是粘贴完自动发生的

## 适合谁

- 想要一个启动快的笔记工具，不想等 app 加载
- 笔记散落在各个平台，想集中到一个地方
- 经常在没信号的地方（飞机、地铁）需要记东西
- 不太信任厂商锁定数据，想要本地优先、随时能导出的工具
- 觉得 notion 太重、备忘录太简陋，中间缺一个刚好的

## 功能列表

### 编辑器
- 块类型：段落 / 标题 H1-H3 / 待办 / 引用 / 分割线 / 装饰分割线
- 卡片类型：音乐卡片 / 链接卡片 / 图片卡片 / 图文卡片 / 图文行 / 图片画廊
- 斜杠命令面板：20+ 命令、拼音首字母搜索、最近使用
- 智能前缀：`#` `##` `###` `>` `[ ]` `---`
- Markdown 自动渲染：表格 / Callout / 代码块 / 列表 / 引用 / 删除线 / 高亮 / 标签
- 撤销/重做：三指手势 / Cmd+Z+Y / 菜单按钮

### 存储
- IndexedDB 主存储（笔记 + 附件 + 渲染缓存）
- localStorage 双重备份
- Google Drive 云备份（覆盖 + 合并）
- ZIP 导出/导入（含附件）

### 列表
- 时间分组（今天 / 昨天 / 7天 / 30天 / 月份）
- 置顶
- 图片缩略图自动提取
- 标签自动提取显示
- 展开/折叠长笔记

### 个性化
- 字号：小/中/大
- 笔记专属背景色：8 种预设
- 笔记头像：自定义图片

## 技术相关

- React 18（UMD，没有构建工具）
- contentEditable 编辑器
- IndexedDB + localStorage 分层存储
- Cloudflare Worker 抓链接元数据
- Google Drive API（可选）
- fflate / marked / DOMPurify
- 所有依赖本地化，不需要 CDN

### 模块结构

```
Moments.html        ← 主应用壳
src/
  constants.js      ← 配置
  utils/            ← 工具函数
  services/         ← 存储层
  blocks/           ← 13 种块
  editor/           ← 编辑器核心
  components/       ← UI 组件
  hooks/            ← 手势 Hook
vendor/             ← 本地依赖
```
